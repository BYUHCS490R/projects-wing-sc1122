block tags

1. block tags are tags that starts on new line and with the whole width. Inline tags are tags that will stay with other tags in the same line.

2. margins are spaces that outside of the box. border is a line that surround the box. Padding is the spaces inside the box.

3. we mainly use pixel(px) but there are lots of unit we can use. like cm,mm,pt,and %.

4. when there is a image. since image is inline tag. so when we want the left amd right side without other content we can use figure to make it as a block tag.
